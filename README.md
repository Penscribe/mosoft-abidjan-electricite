# ⚡ Mosoft Abidjan Électricité

Plateforme participative de signalement des coupures d’électricité dans le District Autonome d’Abidjan.

---

## 🌍 Objectif

Ce projet vise à créer un système citoyen de remontée d’informations sur les coupures d’électricité, basé sur la participation des habitants.

Il permet de :
- Signaler une coupure
- Confirmer une coupure existante
- Signaler un rétablissement
- Visualiser l’état des zones en temps quasi réel

---

## 🗺️ Zones couvertes

Le système couvre les principales communes d’Abidjan :
Abobo, Yopougon, Cocody, Plateau, Marcory, Treichville, Koumassi, Adjamé, Attécoubé, Port-Bouët, Bingerville, Songon, Anyama.

---

## ⚙️ Fonctionnalités

- Carte interactive (Leaflet)
- Signalement citoyen des coupures
- Système de couleurs :
  - 🟢 Vert : normal
  - 🟠 Orange : suspicion de coupure
  - 🔴 Rouge : coupure confirmée
  - ⚪ Gris : zone inactive
- Historique des événements
- Graphique des coupures
- Simulation de signalements

---

## 🧠 Principe de fonctionnement

Les données sont actuellement stockées localement via `localStorage`.

> Version actuelle : prototype frontend (sans backend)
